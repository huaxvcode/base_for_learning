# 移动构造函数

移动构造函数，使用 `std::move` 移动过去后，就会立即被析构删除掉！

```c++
struct Node
{
	int* ptr;
	// 构造函数
	Node() : ptr(new int(314)) {} 
	// 拷贝构造函数
	Node(const Node& t) { ptr = new int(*t.ptr); }
	// 移动构造函数
	Node(Node&& t) {
		ptr = t.ptr; 
		t.ptr = nullptr; // 这句话一定要！！！
	}
	// 析构函数
	~Node() { delete ptr; }
};

Node get_node() {
	return Node();
}

void solve() {
	Node a = std::move(get_node());
	std::cout << *a.ptr << '\n';
	Node b = std::move(a);
	std::cout << *b.ptr << '\n';
}
```
