# 断言

语法：

```java
assert 条件: 输出内容;
```

或者：

```java
assert 条件;
```

**如果条件为 `false` 时，就会断言**，非常方便调试

默认情况下，断言功能是关闭的，在运行时可以添加 `-ea` 选项启用断言功能：

```bash
javac Main.java
java -ea Main
```
