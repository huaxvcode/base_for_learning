# 覆盖方法



```java
class A {
    public void print() {
        System.out.println("this is the class A");
    }
}

class B extends A {
    public void print() {
        super.print();
        System.out.println("this is the class B");
    }
}

public class Main {
    public static void main(String[] args) {
        var op = new B();
        op.print();
        A op_ = new B();
        op_.print();
    }
}
```
