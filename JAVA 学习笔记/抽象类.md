# 抽象类

被 `abstract` 修饰的类都是抽象类，

被 `abstract` 修饰的方法都是抽象方法，抽象方法不需要具体化

示例：

```java
abstract class Name {
    String name;
    abstract void showName();
}

class A extends Name {
    { name = "A"; }
    void showName() {
        System.out.println("My name is: " + name);
    }
}
class B extends Name {
    { name = "B"; }
    void showName() {
        System.out.println("My name is: " + name);
    }
}

public class Main {
    public static void main(String[] args) {
        Name ta = new A();
        Name tb = new B();
        ta.showName();
        tb.showName();
    }
}
```

代码输出：

```
My name is: A
My name is: B
```
