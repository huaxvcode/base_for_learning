# java 的基本浮点类型

- `float` $4$ 字节，有效位数：6 到 7 位
- `double` $8$ 字节，有效位数：15 位

**float 字面量：**

在 java 中，默认情况下所有小数字面量都属于 double 字面量，

只有末尾带 `F` 或者 `f` 的都属于 float 字面量
